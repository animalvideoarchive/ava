Introducing ZVDB

Empower Zoo Research With Intelligent Video Insight

The Zoological Video Database makes it easy to apply advanced media intelligence to video libraries of wildlife behavior for more connected and collaborative science at zoos and partner institutions worldwide.

Why ZVDB?

Designed For Zoological Research At Scale

ZVDB bridges the gap between high-volume video archives and actionable science. Built alongside leading zoos, it simplifies the discovery of relevant footage, standardizes metadata capture, and ensures long-term stewardship through automated lifecycle policies. From curator dashboards to analysis-ready exports, every touchpoint balances power with ease-of-use.

Guided Metadata

Apply behavioral labels, species taxonomy, and husbandry context with reusable templates. Keep records accurate across teams while reducing manual data entry.

Smart Search

Pinpoint clips by animal, enclosure, time range, or research status using rich filters that query synchronized DynamoDB and AppSync datasets in real time.

Glacier Savings

Automated lifecycle rules move archival footage to Amazon S3 Glacier Deep Archive while keeping preview assets instantly available.

Secure Sharing

AppSync-backed permissions and download carts streamline approvals, ensuring sensitive media stays within trusted teams and partner institutions.

Architecture Snapshot

Cloud-Native From Camera Trap To Curated Collection

ZVDB combines an AWS CDK-driven backend with a React + Apollo Client frontend. Together they deliver consistent deployments, scalable performance, and a best-in-class user experience for animal behavior specialists.

AWS Foundations

  • AppSync GraphQL API orchestrates queries and mutations.
  • Lambda functions handle video ingest, tagging, and presigned URL delivery.
  • DynamoDB stores users, videos, collections, and download requests.
  • S3 lifecycle policies shift footage to Glacier Deep Archive automatically.

Frontend Experience

  • React SPA with Material UI and thematic context switching.
  • Apollo Client caches AppSync responses for instant browsing.
  • Upload manager supports simultaneous large video transfers.
  • Dynamic dashboards expose request KPIs and curation insights.

Operational Confidence

  • GitHub workflows automate CDK deployments and frontend releases.
  • Terraform provisions static hosting with CloudFront edge delivery.
  • Cognito secures identity, while IAM fine-tunes service access.
  • Observability via CloudWatch and structured logging in Lambda.
