Why AVA?

Designed For Zoological Research At Scale

AVA bridges the gap between high-volume video archives and actionable science. Originating at the Saint Louis Zoo, AVA simplifies the discovery of relevant footage, standardizes metadata capture, and ensures long-term stewardship through automated lifecycle policies.

Guided Metadata

Apply behavioral labels, species taxonomy, and husbandry context with reusable templates. Keep records accurate across teams while reducing manual data entry.

Smart Search

Pinpoint clips by animal, enclosure, time range, or research status using rich filters.

Glacier Savings

Automated lifecycle rules move archival footage to Amazon S3 Glacier Deep Archive while keeping preview assets instantly available.

Secure Sharing

AppSync-backed permissions and managed collections streamline approvals, ensuring sensitive media stays within trusted teams and partner institutions.

Architecture Snapshot

Cloud-Native From Camera Trap To Curated Collection

AVA combines an AWS CDK-driven backend with a React + Apollo Client frontend. Together they deliver consistent deployments, scalable performance, and a great user experience for animal behavior specialists.

AWS Foundations

  • AppSync GraphQL API orchestrates queries and mutations.
  • Lambda serverless functions handle video ingest, tagging, and delivery.
  • DynamoDB stores users, videos, collections, and requests.
  • S3 lifecycle policies shift footage to Glacier Deep Archive automatically.

Frontend Experience

  • React app with Material UI and thematic context switching.
  • Apollo leverages AWS AppSync for instant browsing.
  • The upload manager supports large video transfers.
  • Alternative low-cost migration options and integrations are in active development.

Operational Confidence

  • GitHub workflows automate CDK deployments and frontend releases.
  • Terraform provisions static hosting with CloudFront edge delivery.
  • Cognito secures identity, while IAM fine-tunes service access.
  • Observability via CloudWatch and structured logging in Lambda.
